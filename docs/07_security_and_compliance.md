# Security and Compliance

## GDPR Principles
- Data minimization
- Right to deletion
- Clear data ownership

---

## COPPA-Lite Approach
- Parents control accounts
- No direct communication between kids
- No ads or tracking

---

## Data Storage Rules
- Avoid birthdays
- Avoid real names when possible
- Store only what is required for gameplay

---

## Copilot Prompt
Default to minimal data collection.
Avoid sensitive personal data.
